关键设计决策

性能保障

  • 波形 100ms/帧 → uPlot 命令式 data.dataset[0] 直接更新，不触发 React render
  • 低频 7s/帧 → Observable store → subscribe 到单个 Chart 组件
  • 状态事件 → React state，无条件覆盖 UI
  • 所有图表卡片: contain: layout style

需要注意

  • uPlot 需要 ResizeObserver 手动计算高度
  • Ring Buffer 需预分配 TypedArray 避免 GC
  • 多设备切换需清理上一设备的订阅
